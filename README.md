# StepSystem

Stepsystem is an all in one study system for students.

## Features:

- A feel good study system.
- Has (or will have) a lot of study tips for you to choose and experiment.
- If my system doesn't work, you can make your own.
- Abbrivations and acronyms for the name of the systems to make it a bit more fun!
